# Howto

Start with creating the module in the current folder:

```bash
go mod init apiWithDocker
```

Install the dependency to `github.com/gin-gonic/gin`:
`-u` means update.

```bash
go get -u github.com/gin-gonic/gin
```

Create your main go file.
